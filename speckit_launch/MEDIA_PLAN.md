# 🎬 SpecKit Media & Demo Plan

To capture the "Magic" of SpecKit, we need high-quality visual assets for the GitHub README and social media.

## 1. Hero GIF: "From Idea to Repo"
- **Scene**: A terminal window.
- **Action**: 
  1. User types `speckit run "I want a simple task tracker"`.
  2. The Progress bar appears.
  3. The "Product Milestone" descriptions flash by (Drafting, Designing, Hardening...).
  4. The Success table appears.
  5. User types `ls -R out/` to show the full repo structure.
- **Goal**: Show the speed and completeness.

## 2. Dashboard Screenshot: "The Brain"
- **Scene**: High-contrast terminal (Dracula or One Dark theme).
- **Action**: Running `speckit dashboard`.
- **Goal**: Highlight the professional DAG visualization and LLM usage tracking.

## 3. Dry-Run Diff: "Safety First"
- **Scene**: Split-screen terminal.
- **Action**: Running `speckit run "..." --dry-run`.
- **Goal**: Focus on the red/green `git diff` output to show developers we are safe to use.

## 🎨 Asset Guidelines
