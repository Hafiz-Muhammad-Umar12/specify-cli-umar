# 🗺️ SpecKit Roadmap

Our goal is to move from **assisted coding** to **autonomous engineering**.

## 🎯 Phase 1: Foundation (Current)
- [x] DAG-based execution engine.
- [x] Multi-agent collaboration (Arch, Sec, Dev, QA, Rev).
- [x] SQLite state machine for pause/resume.
- [x] Basic TDD loops and Git integration.

## 🚀 Phase 2: Professionalization (Next)
- [ ] **Native IDE Integration**: Real-time DAG visualization inside VS Code / Cursor.
- [ ] **Advanced RAG**: Integration with external docs (FastAPI, React, etc.) for up-to-date coding standards.
- [ ] **Cloud-Native Deployment**: Automatic generation of Dockerfiles, K8s manifests, and GitHub Actions.
- [ ] **Frontend Support**: Improved specialized agents for React/Vue component generation and CSS styling.

## 🌟 Phase 3: Autonomous Scale
- [ ] **Multi-Repo Orchestration**: Ability to build and coordinate microservices across separate repositories.
- [ ] **Self-Hosting**: Capability to run local models (Llama 3) for the entire pipeline.
- [ ] **Collaborative Agents**: Support for multiple humans reviewing different parts of the DAG simultaneously.
